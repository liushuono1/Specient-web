# 行业页面：设计领域 (Design)

## 1. 页面定位
强调**创意与规范的平衡、辅助提效**。展示Agent如何处理重复性工作，让设计师回归创意。

## 2. 文案与内容

### A. 头部 (Hero)
*   **标题**: 规范即设计 (Spec is Design)
*   **副标题**: 告别繁琐的切图与标注。让设计规范(Design System)直接驱动生产流程。
*   **背景素材**: 极简主义的黑白线条构成的建筑草图，随着光标移动自动填充材质和光影，变成逼真的3D渲染图。

### B. 功能应用

#### 应用一：设计规范驱动的流程 AI 化 (Spec-Driven Workflow)
*   **描述**: 设计师只需维护 Design Token 和 组件库规范。Agent 自动根据需求文档生成符合规范的高保真 UI 界面。
*   **素材描述**: 
    *   **视频**: 屏幕左边是一段 JSON 格式的设计规范（颜色、间距、圆角），右边是一个混乱的页面。随着左侧 JSON 被修改，右侧页面瞬间自动调整布局，变得井井有条。

#### 应用二：AI 辅助设计 (AI Copilot for Designers)
*   **描述**: 素材检索、色彩搭配建议、批量图层重命名、多尺寸适配。
*   **素材描述**: 
    *   **UI截图**: 在 Photoshop 或 Figma 界面侧边栏中，悬浮着 SpecAgent 的插件窗口。设计师输入“生成 10 张不同场景的 Banner”，插件窗口飞速滚动生成预览图。

## 3. 样式要求
*   **色彩**: 炫彩 (Holographic) 或 黑白灰极简搭配高饱和度酸性色彩 (Acid Graphics)。
*   **字体**: 无衬线几何字体（如 Helvetica, Futura），体现现代感。
*   **布局**: 瀑布流 (Pinterest 风格) 展示设计案例。

## 4. 数据来源
*   本页面内容由 `design_data.json` 驱动渲染。
